# Projekt Task manager

Jednoduchá konzolová aplikace v Pythonu pro správu úkolů.  
Umožňuje přidávat, zobrazovat a mazat úkoly prostřednictvím textového menu.

---

## Funkce aplikace

- **Přidání úkolu** – uživatel zadá název a popis úkolu.
- **Zobrazení úkolů** – vypíše všechny uložené úkoly.
- **Odstranění úkolu** – smaže úkol podle jeho pořadového čísla.
- **Hlavní menu** – navigace mezi funkcemi.
- **Ošetření chyb** – prázdné vstupy, neplatné volby, nečíselné hodnoty.
